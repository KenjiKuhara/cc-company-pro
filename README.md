# CC Company Pro

> **Pro = プロイノベーション**。株式会社プロイノベーションが提供するCC Company拡張版です。

仮想会社組織を構築・運営する Claude Code プラグイン。
[Shin-sibainu/cc-company](https://github.com/Shin-sibainu/cc-company) をベースに、**情報システム部（AI推進マネージャー）** と **コンテンツ自動蓄積機能** を追加しています。

## こんな人におすすめ

- ChatGPTは使っているけど、IDE（Cursor、VS Code、Claude Code）はまだ触っていない
- AIを使いこなしたいけど、何から始めればいいか分からない
- **SNSやnoteで発信したいけど、続かない**
- 一人社長、または少人数のチームで人手が足りない

## CC Company Proの3つの強み

### 1. 仕事するだけでSNS発信とnote記事が勝手にできる

日々の仕事を秘書経由で進めるだけで、コンテンツの種（content-seeds）が自動的に記録されます。
「何をやったか」「なぜそうしたか」「どう解決したか」が会話の副産物として蓄積。
溜まったら記事化やSNS投稿文の生成もできます。

- 意識してメモを取る必要なし。普通に仕事するだけ
- ChatGPTでやっている仕事を、ツールを変えるだけで発信までできる
- **追加の努力ゼロでコンテンツが生まれる**

### 2. 気づいたらAI活用のマニュアルができている

SNS発信の副産物として、AI活用のナレッジも自然に蓄積されます。

- 「この業務でAIをどう使ったか」が記録される
- 判断基準の変化の経緯が追える（いつ、なぜ変更したか）
- 後から見返せばマニュアルになる。途中から入った社員にも分かる
- 問題が起きたタイミングで判断基準をアップデートできる

### 3. 情報システム部がAI活用の実践事例を自然に紹介

情報システム部（AI推進マネージャー）が秘書との会話を常に聞いています。
関連するAI活用の実践事例（久原健司のnote記事 https://note.com/kenji192 ）があれば、秘書経由で自然に紹介します。

- 「こんなことしたい」と思ったとき、実践している人の事例が出る
- 困ったときに聞ける人間がいる安心感
- 明示的に依頼すれば、IT/AIの専門コンサルティングとして全力対応

## 基本機能

### 秘書が窓口
- `/company` コマンドで秘書に話しかけるだけ
- TODO管理、壁打ち、相談、何でも対応
- 部署を意識する必要なし。秘書が適切に振り分け
- **会話の中身がコンテンツの種として自動記録される**

### 常設部署
| 部署 | フォルダ | 役割 |
|------|---------|------|
| 秘書室 | secretary | 窓口・相談役・タスク管理・コンテンツの種の記録 |
| CEO | ceo | 意思決定・部署振り分け |
| レビュー | reviews | 週次・月次レビュー |
| 情報システム | it | AI推進マネージャー・IT/AI相談窓口 |

### 選択可能な部署
PM、リサーチ、マーケティング、開発、経理、営業、クリエイティブ、人事から選択。カスタム部署の追加も可能。

## インストール

Claude Code のプラグインマーケットプレイスからインストール:

```
/install cc-company-pro
```

または手動でリポジトリをクローン:

```bash
git clone https://github.com/KenjiKuhara/cc-company-pro.git
```

## 使い方

1. Claude Code で `/company` と入力
2. 秘書がオンボーディングを開始（事業内容、目標、部署選択）
3. `.company/` フォルダが自動生成される
4. 以降は `/company` で秘書にいつでも相談可能
5. 仕事を進めるだけで、コンテンツの種とAI活用ナレッジが溜まっていく

### note記事インデックスの更新

情報システム部が参照するnote記事インデックスは以下で更新できます:

```bash
python3 scripts/note-index-update.py
```

cron で毎日自動更新する場合:

```bash
crontab -e
# 以下を追加:
0 9 * * * /path/to/cc-company-pro/scripts/note-index-cron.sh >> /tmp/note-index.log 2>&1
```

## 動作イメージ

**日常の仕事で:**

ユーザー: 「今日の打ち合わせの内容をまとめて、次のアクションを整理してほしい」

秘書: 「承知しました！打ち合わせの内容を整理しますね。」
→ TODO整理 + コンテンツの種に「打ち合わせの課題→解決プロセス」を自動記録

**AI関連の相談で:**

ユーザー: 「AIツールの導入を検討しているんだけど、何から始めればいい？」

秘書: 「AI導入ですね！まずは現在の業務フローを整理するところから始めましょう。あ、そういえばこの件、久原さんがnoteで似たようなテーマで書いていたのを見つけました。参考になるかもしれません。」

**直接依頼で:**

ユーザー: 「情報システム部に相談。社内のセキュリティポリシーを見直したい」

秘書: 「情報システム部に確認しますね。」
→ 情報システム部がIT/AIの専門知識を駆使して全力対応
